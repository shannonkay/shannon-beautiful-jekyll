Live Journal Mood Themes
'Beauty and the Beast' mood theme
[by spiffydaze]

Instructions:
note: all links open in new windows

Log into Live Journal. Then once you have downloaded the mood icon set, upload the pictures to your server. Several sites that promote free image hosting include Web Post, Photobucket, ImageStation, and Picture Trail.

Now, open the Admin Console.

In the text box type moodtheme_create "Beauty and the Beast" "Beauty and the Beast" and then click the 'execute' button.

Once that's done you'll see a message which reads: 'Success. Your new moodthemeid=#.' Make sure you either copy that number or write it down. Don't close the window.

Now open moodtheme.txt

On every line there is a '1234'. Change that to the number you've written done. An easy way to do this is using the 'replace' function in your text editor (CTRL+H). Then make sure you've changed the URL to the appropriate location of the images on your server. For example, I might change it to:
'http://sarcastic-muse.com/ljmoodicons/'.

Now copy the entirety of that file ('select all'). Copy this into the box in your open browser window. Click 'execute'.

If done sucessfully you should see multiple lines of 'data inserted'. If not, just go through the stages again, and check that you are successfully logged into Live Journal.

Now, you can modify your journal settings. In the drop down box under 'Mood Icons' choose 'Beauty and the Beast' (it's at the bottom).

Save your changes, and that's it!
